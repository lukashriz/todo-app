# Todo App 💚

A simple, clean, and functional **Todo List Application** built with **HTML, CSS, Tailwind CSS, and vanilla JavaScript**. This app uses **localStorage** to persist your tasks even after refreshing the browser or closing the tab.

![Todo App Screenshot](https://placehold.co/600x400/10b981/ffffff?text=Todo+App+Preview)

## ✨ Features

- **Add new tasks** with input validation
- **Mark tasks as completed** (with visual strikethrough)
- **Delete individual tasks**
- **Clear all completed tasks** at once
- **Task counter** showing total and completed tasks
- **Responsive design** – works on mobile, tablet, and desktop
- **Persistent storage** using `localStorage`
- **Light green theme** matching my portfolio design
- **No external dependencies** – pure vanilla JavaScript

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **Tailwind CSS** (via CDN)
- **JavaScript** (ES6+)
- **Font Awesome** (for icons)

## 📦 How to Use

### Option 1: Open Directly
1. Download or clone this repository
2. Open `index.html` in any modern web browser
3. Start adding your tasks!

### Option 2: Host on GitHub Pages
1. Fork this repository
2. Go to **Settings > Pages**
3. Select **main branch** as source
4. Click **Save**
5. Your app will be live at: `https://your-username.github.io/todo-app`

## 💾 Local Storage

All your tasks are automatically saved to your browser's `localStorage`, so they will persist even after:
- Refreshing the page
- Closing the browser
- Restarting your computer

To clear all tasks, simply clear your browser's local storage for this site.
